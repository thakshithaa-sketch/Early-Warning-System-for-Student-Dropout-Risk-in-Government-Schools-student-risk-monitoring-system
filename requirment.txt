# 🎓 Student Risk Monitoring System

## Problem
Many students in government schools drop out due to low attendance, poor academic performance, and socio-economic factors.

## Solution
This project builds an AI-powered Early Warning System that helps teachers identify students at risk of dropping out and suggests interventions.

## Features
- Risk score calculation
- AI dropout prediction
- Government policy recommendations
- Parent communication generator
- Teacher intervention tracking
- Student progress comparison dashboard

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn

## Installation

Clone the repository

git clone https://github.com/your-repo-name

Install dependencies

pip install -r requirements.txt

Run the application

streamlit run app.py

## Impact
Helps schools identify at-risk students early and provide timely support to prevent dropouts.
